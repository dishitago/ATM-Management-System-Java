# ATM-Management-System-Java

A simple console-based ATM simulation built using Java.

## Features
- PIN Authentication
- 3 Login Attempts
- Balance Inquiry
- Cash Withdrawal
- Cash Deposit
- Exit System

## Concepts Used
- Java Basics
- Scanner Class
- Loops
- Conditional Statements
- Menu-Driven Programming

## Future Enhancements
- Change PIN
- Transaction History
- Fast Cash
- Multiple Accounts
